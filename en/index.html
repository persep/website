We are Ruby/Rails developers

web agency for startups with open source software

What we do

Enterprises

We manage very high trafic websites using reliable PaaS technologies to deploy peacefully and scale automatically applications that we have optimized for performance and robustness (caching, background jobs).

Startups

We build MVPs by iterating shortly and guiding you through technological choices for quick prototyping, short time-to-market and activity monitoring until you are successful enough to have your own internal dev team.

Code quality

Our code is versioned (git), organised (gitflow), tested (RSpec, Capybara) within continuous integration (Travis, Codeship), continuous audit (Codeclimate, code reviews) and continuous deployment (Heroku) processes.

Agility

Our terms (in French) are agile development contracts.
And the code we write belongs to you.

Pricing
